===== Настройка локального окружения инженера =====

  * установить основной системой на компьютер современный дистрибутив Linux (например, Ubuntu или CentOS)
  * установить Qt Creator и настроить стиль C++ в соответствии с документом по стилю программирования
    * число отступов
    * расположение фигурных скобок в функциях и классах
  * установить Git Kraken (пробную версию)
    * сделать fork данного репозитория
  * установить Visual Studio Code
  * установить zsh и oh-my-zsh
    *  https://xakep.ru/2017/05/18/cli-console-tips/
  * раздел "Мигрируем на zsh"

===== Критерии выполнения =====
  * создать ветку с названием feature/local-environment
  * экспортировать настройки стиля C++ из Qt Creator и разместить их в каталоге styles
  * выполнить коммит
    * см. рекомендации сообщения к коммиту в Pro Git
  * выполнить pull request
